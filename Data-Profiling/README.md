# 业务场景模拟器

输入一个行业名称和探查深度，自动生成该行业的业务场景模拟网页，包含业务线清单、业务流程、表结构 DDL、模拟数据、ER 关系图。

## 快速开始

### 方式一：直接在浏览器中使用（推荐）

1. 双击打开 `src/index.html`
2. 输入行业名称，选择深度
3. 点击"生成"，页面会显示生成进度

### 方式二：AI 预生成离线文件

告诉 AI：`帮我生成电商行业的业务场景，深度2`

AI 会调用 API 生成数据，注入模板后输出 `dist/biz-scenario-{行业}-{时间戳}.html`，可离线查看。

## 页面说明

生成后的页面包含四个 Tab：

| Tab | 内容 |
|-----|------|
| 业务流程 | 流程描述 + Mermaid 流程图 + 涉及的表 |
| 表结构 | 字段清单表格 + DDL 代码块（可复制） |
| 模拟数据 | 可滚动数据表格 + INSERT 语句（可复制） |
| ER 图 | 当前业务板块的表间关系图（含主外键连线） |

左侧导航支持展开/收起业务线和流程，点击流程切换内容。

## 技术栈

- 纯 HTML/CSS/JS，无构建工具
- Mermaid.js（流程图 + ER 图）
- highlight.js（DDL 代码高亮）
- OpenAI 兼容 API（数据生成）
